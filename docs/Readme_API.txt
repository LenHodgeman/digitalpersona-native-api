DigitalPersona Access Management API is a library that allows you to authenticate users in your applications. 

This document provides late-breaking or other information that supplements the DigitalPersona Access Management API documentation 
http://www.crossmatch.com/Support/Reference-Material/DigitalPersona-Altus-Reference-Material/


-------------------------
How to Use This Document
-------------------------

To view the ReadMe file on-screen in Windows Notepad, maximize the Notepad window. On the Format menu, click Word Wrap. To print the ReadMe file, open it in Notepad or another word processor, and then use the Print command on the File menu.


-----------------
1. COMPATIBILITY
-----------------

DigitalPersona Access Management API is compatible with 
 - DigitalPersona 2.1.0


-----------------------
2. SYSTEM REQUIREMENTS
-----------------------
System requirements are described in Developers Guides, located in Docs folder.

In order to use DigitalPersona Access Management API,

 - DigitalPersona Workstation 2.1.0 should be installed on supported OS 
 - DigitalPersona Web server components should be installed and IIS server should be configured in order for DigitalPersona Web SDK Samples including DPWebDemo to work. 

---------------------------
3. CONTENET OF THIS PACKAGE
---------------------------
1. DigitalPersona Web server components installation.
2. DigitalPersona Android AUTH SDK  - requires the DigitalPersona Web server components.
3. DigitalPersona C++ and .Net AUTH SDK.
4. DigitalPersona Java AUTH SDK  - requires the DigitalPersona Web server components.
5. DigitalPersona Web SDK Samples including DPWebDemo - requires the DigitalPersona Web server components.
6. Documentations.
